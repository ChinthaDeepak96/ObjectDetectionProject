# Object Detection with OpenCV for Autonomous Driving

This project detects pedestrians, vehicles, and traffic signs using OpenCV.

## Setup
1. Install Python and dependencies.
2. Run `python src/main.py`.

## Features
- Face detection with Haar Cascades.
- Car and pedestrian detection.
- Video processing.
